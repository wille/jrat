# jRAT Controller

Remote Administration Tool written in Java supporting all major operating systems (Windows, OS X, Linux & BSD), desktop computers and servers

## Arguments

| Argument	  	| Description
| ---         	| :---
| --debug		| Debugging mode
| --genkey		| Create new key file to default location (files/jrat.key)
| --hidetitle	| Main frame title is only "jRAT"
| --showhexkey	| Prints current key file to console
